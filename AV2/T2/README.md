## Trabalho Pratico 2 - Grafo do Cavalo (3x3)

Implementacao em Python do T2 da disciplina de Grafos.

## Estrutura

- dados/entrada.txt: grafo no formato algs4 (V, E, arestas)
- src/main.py: ponto de entrada
- src/graph.py: representacao por lista de adjacencia
- src/cc.py: componentes conexas
- src/cycle.py: deteccao de ciclo com vertices de um ciclo encontrado

## Como executar

1. Abra o terminal em AV2/T2
2. Rode:

	python -m src.main

## O que o programa responde

- Lista de adjacencia do grafo
- Quantidade de componentes conexas e vertices de cada componente
- Distancia minima entre (0,0) e (2,2)
- Se o grafo possui ciclo
- Complexidade de tempo e espaco do algoritmo de ciclo
- Vertices de um ciclo encontrado (quando existir)
